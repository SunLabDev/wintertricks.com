About

wintertricks.com

Winter Tricks is an open source project by the Winter CMS community.

The source of this website is available on GitHub, licensed under an MIT license.

This website aims to be an easy to understand resource that provides the best solutions to every-day problems that a developer using Winter CMS might encounter.

The open source nature of the website enables the community to further improve the site over time.
